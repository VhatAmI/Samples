18 /*******
19  *
20  * SpokesWrapper.cpp/.h
21  *
22  * SpokesWrapper.cpp/.h is a wrapper around the Plantronics Spokes COM Service API for C++.
23  *
24  * It's purpose is to make it easier and simpler to integrate support for Plantronics devices into any applications.
25  *
26  * It achieves this by hiding a lot of the more tricky aspects of integration behind the wrapper and presenting
27  * a simple and consistent set of Event Handlers and functions for the core features of the SDK that the user
28  * will typically be needing.
29  *
30  * *** WARNING !!! This source code is provided *As Is*! Is is intented as a sample code to show ways of integrating
31  * with the Spokes "COM Service API". However in case of problems please feel free to contact Lewis Collins
32  * directly via the PDC site at this address: http://developer.plantronics.com/people/lcollins/ ***
33  *
34  * The latest version of this file will also be maintained (and feel free to create your own Fork!) on Github, here:
35  *
36  * https://github.com/pltdev/Samples/tree/master/wrappers
37  *
38  * Read more about Plantronics Spokes at the Plantronics Developer Connection web site:
39  *
40  * http://developer.plantronics.com/community/devzone/
41  *
42  * Lewis Collins, http://developer.plantronics.com/people/lcollins
43  *
